# KunstTec

Landing page for KunstTec — universal, simple crossplatform apps.

## Deploy to Vercel

1. Push this repo to GitHub (or connect via GitLab/Bitbucket).
2. Go to [vercel.com](https://vercel.com) → New Project.
3. Import the repo and deploy. No build step needed — it's a static site.
4. Vercel will serve `index.html` at the root.

Or use the Vercel CLI:

```bash
npx vercel
```

## Local development

Open `index.html` in a browser, or run a simple server:

```bash
npx serve .
# or: python -m http.server 8000
```
